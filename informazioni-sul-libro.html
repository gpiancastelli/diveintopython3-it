Informazioni sul libro

Il testo di Dive Into Python 3 è distribuito secondo i termini della licenza Creative Commons Attribuzione-Condividi allo stesso modo 3.0 Unported.

La libreria chardet citata nel capitolo Caso di studio: convertire chardet verso Python 3 è rilasciata sotto licenza LGPL 2.1 o successiva. Il risolutore di alfametica citato nel capitolo Uso avanzato degli iteratori si basa sul risolutore scritto da Raymond Hettinger per Python 2, che l’autore ha gentilmente ridistribuito sotto licenza MIT in modo che io potessi convertirlo verso Python 3. I capitoli Uso avanzato delle classi e Nomi dei metodi speciali contengono parti di codice provenienti dalla libreria standard di Python, rilasciata sotto licenza Python Software Foundation License versione 2. Tutti gli altri esempi di codice sono mio lavoro originale e sono distribuiti sotto licenza MIT. I termini completi della licenza sono inclusi in ogni file sorgente.

Gli effetti di evidenziazione dinamica nell’edizione online del libro sono costruiti a partire da jQuery, che è distribuita con una doppia licenza secondo i termini delle licenze MIT e GPL.

L’edizione online si carica così velocemente perché

  1. il codice HTML è compresso.
  2. Gli script e i fogli di stile sono rimpiccioliti dallo YUI Compressor.
  3. Gli script sono combinati per ridurre il numero delle richieste HTTP.
  4. I fogli di stile sono combinati e inseriti in linea per ridurre il numero delle richieste HTTP.
  5. Le proprietà e i selettori CSS non utilizzati sono rimossi sulla base della singola pagina tramite pyquery.
  6. Il caching HTTP e altre opzioni lato server sono ottimizzate sulla base di consigli ottenuti da YSlow e Page Speed.
  7. Il testo usa caratteri Unicode al posto di elementi grafici ovunque possibile.
  8. L’intero libro è stato amorevolmente realizzato a mano in HTML 5 per evitare markup ridondante.
