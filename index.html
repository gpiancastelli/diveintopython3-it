Dive Into Python 3

Dive Into Python 3 tratterà di Python 3 e delle sue differenze rispetto a Python 2. Paragonato al Dive Into Python originale, conterrà circa il 50% di materiale rivisto e il 50% di nuovo materiale. Pubblicherò le bozze online man mano che procedo. La versione finale sarà pubblicata su carta per i tipi di Apress. Il libro rimarrà disponibile online secondo i termini della licenza CC-BY-SA-3.0.

Sono disponibili un changelog, una feed, e una discussione su Reddit. Durante la scrittura, potete scaricare il libro clonando il repository Mercurial: 

you@localhost:~$ hg clone http://hg.diveintopython3.org/ diveintopython3

La versione finale sarà scaricabile in HTML e PDF.
