Dive Into Python 3

Dive Into Python 3 tratta di Python 3 e delle sue differenze rispetto a Python 2. Paragonato a Dive Into Python, contiene circa un 20% di materiale revisionato e un 80% di materiale nuovo. Sto pubblicando le bozze online man mano che procedo. Per favore, inviatemi feedback. La versione finale sarà pubblicata su carta per i tipi di Apress. Il libro rimarrà disponibile online secondo i termini della licenza CC-BY-SA-3.0.
