Immersione in Python 3

Immersione in Python 3 è la traduzione italiana di Dive Into Python 3, un libro scritto da Mark Pilgrim e pubblicato da Apress nel 2009 che parla di Python 3 e delle sue differenze rispetto a Python 2. Paragonato a Immersione in Python, questo libro contiene circa un 20% di materiale revisionato e un 80% di materiale nuovo. Ora la traduzione è completa, ma i commenti sono sempre benvenuti.
