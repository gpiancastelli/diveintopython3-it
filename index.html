Dive Into Python 3

Dive Into Python 3 tratterà di Python 3 e delle sue differenze rispetto a Python 2. Paragonato al Dive Into Python originale, conterrà circa il 50% di materiale rivisto e il 50% di nuovo materiale. Pubblicherò le bozze online man mano che procedo. La versione finale sarà pubblicata su carta per i tipi di Apress. Il libro rimarrà disponibile online secondo i termini della licenza CC-BY-SA-3.0.

Potete leggere l'indice completo (non finalizzato) oppure leggere quanto ho scritto finora:

  1. Installare Python
  2. Il vostro primo programma Python
  3. Tipi di dato nativi
  4. Stringhe
  5. Espressioni regolari
  6. Iteratori & generatori
  7. Uso avanzato degli iteratori
  8. Oggetti e orientazione agli oggetti
  9. Test di unità
  10. Programmazione test-first
  11. Refactoring
  12. File
  13. Elaborare documenti HTML
  14. Elaborare documenti XML
  15. Servizi web
  16. Metaclassi
  17. Mettere a punto le performance
  18. Distribuire librerie Python
  19. Creare grafica con la Python Imaging Library
  20. Dove proseguire da qui
  21. Caso di studio: convertire chardet verso Python 3
  22. Convertire codice verso Python 3 con 2to3

Sono disponibili un changelog, una feed, e una discussione su Reddit. Durante la scrittura, potete scaricare il libro clonando il repository Mercurial: 

you@localhost:~$ hg clone http://hg.diveintopython3.org/ diveintopython3

La versione finale sarà scaricabile in HTML e PDF.
